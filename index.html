Gem Game
Counting game using gems

Counting game where a target score will show on the screen and the user is provided with gems with random, but consistent values for the length of the game. The user can find the value of each gem by clicking it and comparing the score before and after clicking the gem. User wins if they hit the target score correctly, and lose if they go over the target score.
